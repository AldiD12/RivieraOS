## 📱 User Experience Flow

**Perfect Event Booking Flow:**
1. User switches to Night Mode (🪩)
2. Sees events list with vibe tags and entry types
3. Filters by vibe (House, Techno, etc.)
4. Clicks event → WhatsApp opens with:
   - Event name and description
   - Venue name and location
   - Date and time
   - Entry type (Free/Reservation/Ticketed)
   - Minimum spend (if applicable)
   - Pre-filled booking request

**Example WhatsApp Message:**
```
Hi! I'd like to book for techno at Beach Bar on Thursday, 12 May 2026.

📅 Event: techno
📍 Venue: Beach Bar
🕐 Date & Time: Thursday, 12 May 2026 at 20:00
💎 Minimum Spend: €200 per table

How many people: 
Preferred arrival time: 
```
